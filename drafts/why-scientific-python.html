Why python instead of Matlab for science?

Mon 25 August 2014

Filed under Misc

Tags science python

Tweet

In my graduate program, Matlab dominates both in coursework and research. Of the four computational neuroscience classes with programing components I have taken, all use Matlab. Two books written by faculty here at U of C use Matlab for code examples and one even has Matlab in the title. I believe every student in my program, except me, is using Matlab in their thesis work. Clearly, for electrophysiology and neuroscience data analysis, Matlab is king.

So why should scientists use Python?
